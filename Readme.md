Ecobici for Firefox OS
=====================

Ecobici for Firefox OS is a mobileApp to locate Ecobici bike's stations

Used technologies
---------------------

- Firefox OS
- Google Maps API v3
- Mootools.js
- Gaia Building blocks
- CSS3 transforms

Required permissions
---------------------

- geolocation
- desktop-notification
